# Périmètre produit — Deal Hunter AI

Ce dépôt porte le produit **Deal Hunter AI / Radar Telegram**.

Le bot et le dashboard sont dédiés à :

- création de radars privés par utilisateur ;
- scans de sources d'annonces et catalogues ;
- alertes deals via Telegram/WhatsApp selon configuration ;
- dashboard Deal Hunter ;
- gestion et suivi des sources ;
- scoring prudent, comparables et suivi des opportunités.

Ce projet ne doit pas être mélangé avec :

- marketplace vendeur ;
- produits boutique ;
- abonnements vendeur ;
- Wallet Pay ;
- PayPal manual ;
- Shop Trust Token ;
- onboarding vendeur ;
- logique VendorGate Marketplace.

Si des restes de marketplace existent dans le dépôt, ils doivent rester séparés et documentés avant toute suppression. Aucune logique vendeur, wallet ou boutique ne doit être introduite dans le bot radar Telegram.
